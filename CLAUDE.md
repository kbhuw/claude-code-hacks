# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Purpose

This repository collects hacks, tips, and commands for Claude Code.

## Structure

- `README.md` - Main tips collection, numbered sequentially
- `scripts/` - Shell scripts and code snippets
- `configs/` - Configuration files (JSON, etc.)

## Adding Tips

When adding new tips:
1. Add under `## Tips` section in README.md as `### Tip N: Title`
2. Number tips sequentially from the last tip
3. Put reusable code in `scripts/` or `configs/` and link from README
4. Link to external resources when relevant
